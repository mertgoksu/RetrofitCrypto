# CryptoCurrencyApp
A basic Kotlin app to practice retrieving data from the Internet with Retrofit/RxJava
<br>

## Features
- Cryptocurrency price display in USD <br>

## Technologies Used
- MVVM structure <br>
- Retrofit <br>
- RxJava <br>
- Activities <br>
- XML <br>
